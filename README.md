# 🚀 Product-Hub - Digital Marketplace Platform

**Product-Hub** একটি আধুনিক এবং রেসপনসিভ রিঅ্যাক্ট অ্যাপ্লিকেশন, যা ডিজিটাল টুলস এবং টেমপ্লেট বিক্রির জন্য একটি ল্যান্ডিং পেজ হিসেবে ডিজাইন করা হয়েছে। এতে ইউজার-ফ্রেন্ডলি ইন্টারফেস এবং ডাইনামিক কার্ট সিস্টেম রয়েছে।

---

## ✨ মূল ফিচারসমূহ (Core Features)

- 🔹 **Interactive Navigation**  
  লোগো এবং রিয়েল-টাইম কার্ট কাউন্ট সহ ক্লিন ন্যাভিগেশন বার।

- 🔹 **Hero Section**  
  আকর্ষণীয় হেডলাইন এবং কল-টু-অ্যাকশন বাটন সহ প্রফেশনাল ব্যানার।

- 🔹 **3-Step Guide**  
  কিভাবে সার্ভিসটি কাজ করে তার সহজ চিত্রায়ন:  
  - Create Account  
  - Choose Products  
  - Start Creating  

- 🔹 **Dynamic Stats**  
  গ্র্যাডিয়েন্ট ব্যাকগ্রাউন্ডে প্রজেক্টের সফলতার পরিসংখ্যান।

- 🔹 **Transparent Pricing**  
  Starter, Pro (Highlighted), এবং Enterprise—এই ৩টি ক্যাটাগরির প্রাইসিং প্ল্যান।

- 🔹 **Smart Product Grid**  
  ক্যাটাগরি ট্যাগ এবং ডিটেইলস সহ কমপ্যাক্ট প্রোডাক্ট কার্ড।

- 🔹 **Integrated Cart System**  
  প্রোডাক্ট অ্যাড করা, রিমুভ করা এবং অটোমেটিক টোটাল প্রাইস ক্যালকুলেশন সুবিধা।

- 🔹 **User Notifications**  
  প্রতিটি অ্যাকশনের জন্য (যেমন: Add to cart, Remove) ইনস্ট্যান্ট টোস্ট মেসেজ।

- 🔹 **Modern Footer**  
  সোশ্যাল লিঙ্ক এবং পলিসি পেজ সহ ডার্ক থিম ফুটার।

---

## 🛠 টেকনোলজি স্ট্যাক (Tech Stack)

- ⚛️ **Frontend Library:** React.js  
- 🎨 **Styling:** Tailwind CSS  
- 🔄 **State Management:** React Hooks (useState, useEffect)  
- 🎯 **Icons:** Lucide React / SVG  
- 🔔 **Notifications:** React-Toastify
